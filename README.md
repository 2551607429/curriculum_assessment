# 基于SSM的高校课程考核管理系统
## 1. 项目描述
1. 此项目为一个基于Vue的前后端分离的基于SSM的高校课程考核管理系统项目
2. 使用了 Vue+NodeJS+ElementUI 等前端新技术 
3. 后端基于Maven多模块 + Spring Boot搭建SSM框架 + MySql
4. 包括学生端、教师端和管理员端 
5. 该部分是基于SSM的高校课程考核管理系统的服务器端
6. 采用模块化、组件化、工程化的模式开发
